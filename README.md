# SujaySarma.Sdk.Azure
Our own SDK to interact with the Azure REST API services
---
I have waited long enough for the developers at Microsoft to create a meaningful and intuitive SDK for Azure. But their [latest efforts](https://github.com/Azure/azure-sdk-for-net/) completely mystifies me. Instead of simplifying things, they are complicating things!

So, this is a renewed effort to create a meaningful, cohesive and intuitive SDK for Azure. I hope to keep this up to date, graceful and meaningful for as long as I can. 

---
